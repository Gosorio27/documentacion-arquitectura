# Arquitectura BP

## Descripción

Este repositorio contiene la documentación técnica y funcional del proyecto, incluyendo arquitectura, diagramas, reglas de negocio y especificaciones de servicios.

## Objetivo

Centralizar y mantener la documentación del proyecto para facilitar la consulta por parte de los equipos de desarrollo, arquitectura, calidad y operaciones.

## Contenido

### Arquitectura
- Documento de Arquitectura Empresarial
- Documento de Arquitectura Técnica
- Diagramas de Componentes
- Diagramas de Despliegue

### Diagramas
- Diagramas de Secuencia
- Diagramas de Flujo
- Diagramas de Integración

### Reglas de Negocio
- Reglas funcionales
- Reglas de validación
- Casos de uso

### APIs y Servicios
- Especificaciones OpenAPI / Swagger
- Contratos REST
- Ejemplos de Request y Response
